# SPEC-XGUI 문서 감사 기록

이 디렉터리는 현재 상태를 분석한 비규범 보고서다. 구현 요구와 계약은 `.moai/specs/SPEC-XGUI-*` 및 `SPEC-XSEAM-002`만 권위로 사용한다.

## 감사 라운드

- `round-1/`: 각 그룹의 초기 적대적 검토 원문. 당시 FAIL/권고는 이후 규범 문서에 반영됐으며 현재 판정으로 사용하지 않는다.
- `audit-v0.4.md`: v0.4 내부 정적 감사. 사용자 승인과 구현 착수 허가를 포함하지 않았으므로 역사 기록이다.
- `round-2.md`: v0.5 전수 구조·추적·평가지표 재감사와 수정 결과.
- `audit-v0.5-final.md`: v0.5 역사 감사. callable·요구사항 집계가 v0.5.1에서 정정돼 현재 판정으로 사용하지 않는다.
- `audit-v0.5.1-final.md`: 현재 기준선 후보의 최종 반복 감사 결과.

현재 기준선 후보는 v0.5.1이며 `approval_state=pending_user`, `implementation_authorized=false`다.
