## 무결성 점검 (v1.0 시점)

- 고아 MR 없음 (MR 10건 전부 PR 연결).
- 고아 PR 없음 (PR 17건 전부 MR·EV 연결).
- 미사용 EV 없음 (EV 16건 전부 1개 이상 PR에서 참조).

## 운용 규칙

1. FR/SWR 전개 시 본 표의 PR 하위에 3-Trace(HW/SW/ME) 체계로 연결.
2. 변경 발생 시 해당 행 기준 상·하류 영향 분석 후 개정.
3. VV/TC 열 충전 완료 시점 = 검증 계획 동결(E3) 게이트.
