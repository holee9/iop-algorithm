# X-ray FPD 영상처리 평가 지표 기준서 (EVAL v1.0 — min/typ/max)

기준 문서: TRM v2.0, PRM v1.0. 본 문서는 알고리즘 개발의 합격선(min)·설계 목표(typ)·과잉설계 상한(max)을 정의한다.

## 0. min/typ/max 정의 및 범위 설정 원칙

| 구분 | 의미 | 설정 근거 |
|---|---|---|
| **min** | 불합격선. 미달 시 해당 기능 릴리스 불가 | 국제 표준 요구치 또는 "무처리 대비 열화 없음" |
| **typ** | 설계 목표. 사양서(PR) acceptance criteria의 기본값 | 상용 제품 실측 문헌의 중앙값 수준 |
| **max** | 과잉설계 상한. 초과 달성 시도는 투자 중단 (수확체감 선) | 문헌상 최고 수준 또는 물리적 한계 근접치 |

범위 폭 보정 원칙: min–typ 간격은 "표준 요구 ↔ 시장 평균", typ–max 간격은 "시장 평균 ↔ 시장 최고"로 설정. 문헌 분포가 좁은 지표는 범위도 좁게, 분포가 넓은 지표(예: DL denoising)는 범위를 넓게 잡았다.

표기: **[B]** = 자기 패널 베이스라인 실측 후 절대값 보정 필요 항목 (상대 지표로 우선 운용).

## 1. 시스템 물리 지표 (검출기+Common Core, IEC 62220-1 / ASTM E2597)

무처리 raw 대비 "알고리즘이 열화시키지 않을 것"이 1차 기준. 절대값은 신틸레이터 종류에 종속.

### 1.1 DQE @ RQA5 (의료) [B]

문헌 분포: CsI 패널 peak DQE 60~64%, GOS 패널 38~44%. 주파수별 참조(CsI급): 1 lp/mm ≈ 0.52, 2 lp/mm ≈ 0.35, 3 lp/mm ≈ 0.21.

| 항목 | min | typ | max |
|---|---|---|---|
| Peak DQE (GOS 패널 기준) | 0.35 | 0.40 | 0.45 |
| Peak DQE (CsI 패널 기준) | 0.55 | 0.60 | 0.65 |
| 보정 알고리즘 적용 후 DQE 변화 | 열화 ≤ 2% | 열화 0% | (개선 주장은 별도 검증 — DQE는 원칙적으로 SW로 상승 불가, 상승 측정 시 측정 오류 의심) |

### 1.2 MTF / SRb [B]

문헌 분포: Nyquist(3.33 lp/mm)에서 CsI ≈ 35%, GOS ≈ 25%.

| 항목 | min | typ | max |
|---|---|---|---|
| 알고리즘 적용 후 SRb 열화 | ≤ 10% | ≤ 5% | 0% (열화 없음) |
| MTF@Nyquist 유지율 | ≥ 90% | ≥ 95% | 100% |

근거: NDT 표준(ISO 17636-2)이 SRb를 shot별 기록 요구 — 필터링에 의한 SRb 훼손은 IQI 판독 실패로 직결되므로 min을 10%로 제한.

### 1.3 Defect / Bad pixel (ASTM E2597 분류 기준)

ASTM E2597은 bad pixel 7종 분류·통계 방법을 정의하나 **합격 수량 표준은 없음** — 응용별 자체 기준 필요. 따라서 아래는 자체 설정치.

| 항목 | min | typ | max |
|---|---|---|---|
| 보정 후 잔존 가시 defect (진단 ROI 내) | 0 (cluster) | 0 | 0 |
| 보정 후 단일 pixel 잔존 가시성 | 관찰자 비인지 | 좌동 | — |
| Defect map 검출 누락률 | ≤ 2% | ≤ 0.5% | 0% |

### 1.4 Lag / Ghost [B]

| 항목 | min | typ | max |
|---|---|---|---|
| First-frame lag (보정 후) | ≤ 5% | ≤ 2% | ≤ 0.5% |
| Ghost 잔상 가시성 (임상 조건) | 비가시 | 비가시 | — |

근거: 기 확보한 forward bias 98% ghost 저감 실적을 typ 근거로 활용. max 0.5%는 문헌상 hardware 보정 상한 수준 — 그 이하 추구는 수확체감.

### 1.5 Line noise (correlated row/column)

| 항목 | min | typ | max |
|---|---|---|---|
| 보정 후 잔존 line artifact 가시성 (균일 조사) | 비가시 (window 표준) | 비가시 (window 확대 2×) | 비가시 (극단 window) |
| 구조물 오염 오보정 발생률 | ≤ 1% (임상/시험 세트) | 0% | 0% |

## 2. Medical Post-processing 지표

### 2.1 노이즈 저감 (고전 + DL)

| 항목 | min | typ | max |
|---|---|---|---|
| SNR 개선 (동일 선량, 무처리 대비) | ≥ +20% | +40~60% | +100% (이상은 과평활 위험) |
| 저선량 동등화질 (선량 절감률 환산) | ≥ 15% | 30% | 50% |
| 구조 보존: SRb 열화 | §1.2 준수 | 좌동 | 좌동 |
| DL 전용: 아티팩트(hallucination) 발생 | 0 (검증 세트) | 0 | 0 |

근거: 선량 절감 typ 30%/max 50%는 scatter correction 임상 검증 수준(SimGrid 약 18.7% DAP 절감, Fuji 팬텀 50% 주장)을 상한 참조로 설정. 50% 초과 주장은 임상 근거가 희소하므로 max로 고정.

### 2.2 Scatter correction / Virtual grid

| 항목 | min | typ | max |
|---|---|---|---|
| 무그리드+보정 vs 물리 grid 화질 (관찰자 동등성) | 비열등 | 동등 | 동등 + 선량 절감 |
| CNR 개선 (무그리드 무보정 대비) | ≥ +20% | +30% | +50% |
| DL scatter 추정 정확도 (MC 기준) [B] | 편차 ≤ 5% | ≤ 3% | ≤ 2% (DSE 수준) |

근거: SimGrid 관찰자 연구(grid 동등 화질, p<0.001)를 typ의 임상 근거로, DSE mean error 1.7%를 max 근거로 설정.

### 2.3 Grid line suppression

| 항목 | min | typ | max |
|---|---|---|---|
| 잔존 grid line 가시성 (표준 window) | 비가시 | 비가시 (확대 포함) | — |
| Grid 주파수 대역 신호 손실 | 국소 손실 허용 | 최소화 | 무손실 (DL 수준) |
| Moiré/aliasing 발생 | 0 | 0 | 0 |

### 2.4 대비 강화 / 관찰자 평가

| 항목 | min | typ | max |
|---|---|---|---|
| 관찰자 평가 신뢰도 (ICC, 판독성 항목) | ≥ 0.7 | 0.75~0.85 | ≥ 0.9 |
| VGA 상대 점수 (경쟁 벤더 처리 대비) | 비열등 | 동등 | 우월 |
| GSDF 적합성 (DICOM PS3.14) | 적합 | 적합 | — (pass/fail 항목) |

근거: MUSICA2/3 임상연구 ICC 0.76~0.86 분포를 typ 범위로 직접 인용.

## 3. NDT Post-processing 지표 (ISO 17636-2 / ASTM E2597)

### 3.1 SNRn / 화질 등급

| 항목 | min | typ | max |
|---|---|---|---|
| SNRn (처리 후, 대표 조건) | Class A 요구치 충족 | ≥ 130 (Class B 대부분 두께 커버) | ≥ 250 (필름 C1급 상회) |
| Duplex wire IQI (SRb) | 표준 요구 wire 판독 | +1 wire | +2 wire (compensation 상한) |
| 단선 IQI 감도 | Class A | Class B | Class B +1 |

근거: SNRn 130이 대부분 두께에서 Class B를 커버(단 극박 두께 예외 — SRb 요구가 IQI wire에 스케일). ISO 17636-2 compensation principle의 wire 보상 상한(2~3 pair)을 max에 반영. 필름 대비 DDA 우위 실증(film SNRn 265 vs DDA 490)이 max 달성 가능성의 근거.

### 3.2 ADR (Gen 2 파일럿)

| 항목 | min | typ | max |
|---|---|---|---|
| 결함 검출 recall (검증 세트) | ≥ 90% | ≥ 95% | ≥ 98% |
| False call rate | ≤ 10% | ≤ 5% | ≤ 2% |
| POD 곡선 a90/95 | 산출 가능 | 인간 검사자 동등 | 인간 우월 |

주: ADR 수치는 문헌 분포(조선소 사례 recall 95.7% 등)를 참조한 잠정치 — 자체 데이터셋 확보 후 보정 필수.

## 4. 성능·운영 지표 (Tier 연동) [B]

| 항목 | min | typ | max |
|---|---|---|---|
| Tier 2 전체 파이프라인 처리시간 (1 frame, full 해상도) | ≤ 3 s | ≤ 1 s | ≤ 0.3 s |
| Tier 1 결정론적 파이프라인 | ≤ 5 s | ≤ 2 s | ≤ 1 s |
| 티어 간 결과 차이 (진단 영향 기능) | 정의된 허용오차 내 | bit 수준 동일 | 동일 |

주: 절대 시간은 사양서 단계 벤치마크 후 확정 — 본 수치는 시장 통상 워크플로(촬영 후 프리뷰 1~3초) 기준 잠정치.

## 5. 운용 규칙

1. **acceptance criteria 없는 PR 금지**: 모든 PR은 본 문서의 지표+조건+min값을 인용해야 리뷰 통과.
2. **회귀시험**: 알고리즘 변경 시 §1 물리 지표 전체 + 해당 post 지표를 골든 데이터셋으로 자동 회귀.
3. **[B] 항목 보정**: 베이스라인 실측(E1 단계) 완료 후 본 문서 v1.1로 절대값 개정.
4. **max 초과 금지 아님**: max는 "달성 금지"가 아니라 "추가 투자 중단 판단선".

## 6. Caveats

- CsI/GOS 이원 기준: 자사 패널 신틸레이터 확정값 기준으로 v1.1에서 단일화.
- ADR·DL 항목 수치는 타 도메인 문헌 참조 잠정치 — 자체 데이터 확보 전 계약·대외 스펙에 사용 금지.
- Bad pixel 합격 수량은 표준 부재로 자체 기준 — 고객/심사 요구 시 응용별 정당화 문서 필요.
- 처리시간(§4)은 해상도·HW 미확정 상태의 잠정치.
