# XDET SWR — Software Requirements Specification v1.2 (상세 사양서 — 전수 딥리서치 + 모듈 아키텍처 원칙 반영)

**근거 등급**: [S] 표준 원문 일치 / [L] 문헌 근거 확보 / [C] 업계 관행 (원문 미대조) / [P] 제안값 (검증 전, 튜닝 대상)

기준: FRD v1.1, EVAL v1.1, 측정 프로토콜 v1.0. 대상 프레임: 3072×3072 / 3072×2560, 16-bit unsigned raw, pitch 140µm (Nyquist f_N = 3.571 lp/mm). 미확정 파라미터는 TBD-[B](1단계 실측 입력) 또는 TBD-[T](튜닝)로 표기. 특허 검토는 릴리스 게이트로 이연(사용자 지시) — 본 사양은 기술 필요성 기준으로 작성하되 회피 검토 대상 절에 ⚠P 표기.

| SWR-1001 | **주파수 설계 전제**: 상용 grid 밀도 30~85 lines/cm = 3.0~8.5 lines/mm, 자사 f_N = 3.571 lp/mm → f_grid > f_N 인 grid(약 36 lines/cm 초과 전부)는 **aliased 주파수 f_a = \|f_grid − k·f_s\| (f_s = 7.143/mm)로 접혀 나타남**. 검출기는 명목 grid 주파수가 아니라 **관측 스펙트럼 피크**를 직접 탐색해야 함 |
