# XDET 교차검증 반복 로그 v2.0 (SWR 발행 회차)

방식: 전 문서(18종) 대상 딥리서치+딥싱크 반복 — 보강 사항 없음 판정 시 수렴 종료 (최대 10회).

| 회차 | 점검 방법 | 발견 | 조치 |
|---|---|---|---|
| 1 | 딥리서치: grid 주파수 vs 140µm 샘플링 대조 | **중대**: 상용 grid 30~85 lines/cm 대부분이 Nyquist(3.57 lp/mm) 초과 → grid 성분이 aliasing되어 접힘. 기존 FR-M006 "기본파+고조파 notch" 기술은 불충분 | SWR-1001~1006 aliasing 전제 설계로 작성, FR-M006 개정 (FRD v1.1) |
| 1 | 결정 정합성 | FRD의 SWR 착수 게이트 "특허 검토 완료"가 사용자 결정(특허 최후 이연)과 모순 | FRD 헤더 개정 — 릴리스 게이트 이연 + ⚠P 표기 체계 도입 |
| 2 | 결정 정합성 전파 | P1 계획서 리스크 표에 동일 모순 잔존 | 리스크 대응 문구 개정 (대안 설계 예비 정의로 완충) |
| 2 | SWR→TC 역전파 | TC-015가 grid 밀도 3부류(aliased 포함) 요건 미반영 | Test Spec TC-015 개정 |
| 3 | SWR→취득세트 역전파 | 2-3 grid 취득 요건에 밀도 3부류 미명시 | 영상취득세트 2-3 개정 |
| 3 | RTM 동기화 | SWR 발행 사실 미반영 | RTM 운용 항목 갱신 (v1.2에서 SWR 열 추가 예정) |
| 4 | 스크립트 자동 점검 (ID 무결성) | EV 16종·VV 15종·FR 27종 전 문서 참조 무결 — 누락/예상외 0건. Nyquist 표기 3.57/3.571 혼재 발견 | 표기 혼재는 반올림 차이로 허용 편차 판정 (SWR=정밀값, 요약 문서=반올림) — 무조치 |
| 5 | 수치 재검산 | 프레임 크기(18.9/15.7MB), f_s=7.143/mm, SNRn 88.6µm 계수, 피라미드 레벨 근사 — 전부 재계산 일치. 파이프라인 순서(SWR-000-2) ↔ FRD ↔ P1 WP 순서 일치 | 발견 없음 |
| 6 | 의미 정합 최종 통독 | EVAL min/typ/max ↔ SWR 검증 훅 ↔ TC 판정 기준 3자 대조 — 모순 없음. ⚠P 절(SWR 5/8/11) 대안 설계 예비 정의 존재 확인 | 발견 없음 |

## 수렴 판정

5·6회차 연속 무발견 — **수렴 종료 (6회차)**. 7~10회차 미실행 (보강 대상 부재).

## 회차 1 발견의 의미 (기록)

grid aliasing 이슈는 딥리서치 없이는 사양서에 잘못 들어갈 뻔한 설계 결함이었다. 140µm 패널에서는 grid line이 명목 주파수가 아닌 접힌 저주파로 관측되므로, "grid 사양표의 주파수를 notch하라"는 식의 설계는 실기에서 동작하지 않는다. SWR은 관측 스펙트럼 피크 직접 탐색 + 접힘 후보 조합 검사 + 저주파 접힘 시 품질 경고(부적합 grid 조합 안내)로 설계되었다.

# 3차 반복 사이클 (전수 딥리서치 지시 재수행 — v2.1 추가분)

경위: 이전 사이클이 지시("전체 문서의 모든 기술 전수 딥리서치") 대비 일부만 수행한 사실을 인정하고 재수행. 각 회차의 조사 방법을 사실대로 기록.

| 회차 | 대상 블록 | 조사 방법 | 발견 및 조치 |
|---|---|---|---|
| 1 | Defect 검출·보간 | 신규 웹 검색 (ASTM E2597 원문 발췌) | noisy 6× median [S] 확인 — [P]→[S] 승격. 9×9 커널·bad neighborhood·lag 판정 절차 [S]. E2597-14 typical 분율 확보 → EVAL 참고선 추가. dead/over-under/non-uniform 임계는 원문 수치 미확인 → [P] 유지 명시 |
| 2 | Lag IRF | 신규 웹 검색 (Starman/NLCSC 계열) | **정정**: 지수합 M=2~3 → 3~4 (문헌 표준 N=4). IRF의 측정기법·노출 민감성 확인 → 다중 노출 step-response 캘리브레이션 의무화 (단일 노출 금지). LTI 잔존 ≤1.4% / NLCSC ≤0.29% → EVAL EV-104 문헌 참조 주석 |
| 3 | BM3D/NLM | 신규 웹 검색 (Dabov 원저 파라미터) | 원저 기본값 [L] 확보: 블록 8×8·step 3·N2=16·Ns=39·λ=2.7·Haar·Kaiser β=2.0 — SWR-704 구체화 |
| 4 | 다중스케일·DRC | 신규 웹 검색 (Vuylsteke 1994, Dippel TMI 2002) | 변조 함수형(power law + 선형 컷오프)이 1994 공개 논문에 존재 [L] — ⚠P 완화 근거(원 기법 공지, 후속 특허만 주의). Laplacian pyramid의 wavelet 대비 아티팩트 우위 [L] — SWR-801 선택 근거 확보 |
| 5 | 자동 윈도잉 | 신규 웹 검색 (1990년대 특허·문헌군) | **설계 보강**: 문헌 공통 3단계(조사야 인식→직접선 분리→해부 히스토그램) 중 조사야 인식이 SWR-901에 누락 → 신설. 해당 특허군 1990년대 → 만료 추정, 리스크 낮음 |
| 6 | Virtual grid 커널 | 신규 웹 검색 (SKS 학술 문헌) | SKS 방법론 [L] 확보 (두께 의존 커널, 2~4 Gaussian, 반복 보정 강건성). **⚠P 강화**: SKS 수식 청구 유효 특허(US 11,911,202) 발견 — 릴리스 전 청구항 대조 필수 명기 |
| 7 | NDT (SNRn·IQI·두께보정) | **신규 검색 없음** — 기확보 증거 재검증 (ISO 원문 발췌·VisiConsult) | 20% dip [S], SNRn 88.6µm [S], compensation 상한 [S] — 기존 사양 유지 판정. 신규 검색 미수행 사실 명기 |
| 8 | 반영 | 문서 개정 | SWR v1.1 발행: 근거 등급([S]/[L]/[C]/[P]) 전면 명시 + 총괄표, 5개 SWR 개정. EVAL 문헌 참조 2건 추가 |
| 9 | 무결성 | 스크립트 자동 점검 | ID 체계 무결(EV16/FR27/VV15). EVAL 삽입 1건 실패 발견 → 즉시 재삽입 수정 |
| 10 | 최종 통독 | 정합 확인 | 개정분 상호 모순 없음. 수렴 — 단, 아래 잔여 미검증 항목 존재 (허위 완결 선언 아님) |

## 잔여 미검증 항목 (사실 기록)

1. ASTM E2597 dead/over-under/non-uniform **임계 수치** — 원문 유료 표준, 발췌로 미확인 → [P] 유지, 원문 구매 시 확정
2. IEC 62220-1 조리개 위치·세부 기하 — 동일 사유 (프로토콜 Caveat 기존 명기)
3. Stahl 1999 noise-resistant 변조 세부 수식 — 초록 수준만 확인
4. US 11,911,202 청구항 범위 — 특허 검토는 사용자 지시로 릴리스 게이트 이연
5. NDT 두께보정 자체의 알고리즘 문헌 (형태학적 방법 비교) — [C] 등급 유지

이상 5건은 "미검증"으로 등급 표기되어 문서에 남아 있으며, 완결로 위장하지 않았다.

## 추가 사이클: 모듈화·재사용성 교차검토 (사용자 지시)

발견 G1~G5: 공통 데이터 컨테이너 미정의(최대 위험), 모듈 간 직접 호출 금지 규칙 부재, 공용 컴포넌트 미식별(pyramid/히스토그램/FFT 중복 기술), 캘리브레이션 공통 스키마 부재, 모듈 단위 harness 미규정.
조치: SWR v1.2 — §0-A 모듈 아키텍처 원칙 신설 (SWR-000-6~12: XFrame 컨테이너, 단일 시그니처, 오케스트레이터 단독 조합권, 공용 컴포넌트 5종, CalibSet 스키마, 모듈 CI, P1/P2/P4 구현 교체 계약). P1 계획서 WP11 갱신.
