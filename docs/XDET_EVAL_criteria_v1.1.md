# X-ray FPD 영상처리 평가 지표 기준서 (EVAL v1.1)

v1.0 대비 변경: ① 기준 패널 확정 — CsI, 140µm pitch, 3072×3072 / 3072×2560 (GOS 기준은 차기 패널용 예약으로 이동) ② 교차검증 리포트 v1.0 보정 반영 (EV-106/205/303 신설, EV-103 재정의) ③ 전 지표 EV ID 부여 ④ 문헌 베이스라인 수치 주입 — 자사 실측은 병행 과제로 강등, [B] 항목은 실측 후 v1.2 치환.

## 0. 기준 패널 및 파생 상수

| 항목 | 값 |
|---|---|
| 신틸레이터 | CsI (needle) |
| Pixel pitch | 140 µm |
| 해상도 | 3072×3072 (9.4MP) / 3072×2560 (7.9MP) |
| Nyquist 주파수 | 3.57 lp/mm |
| 프레임 크기 (16-bit) | 18.9 MB / 15.7 MB |
| 선질 기준 | IEC 62220-1 RQA5 (일반촬영) |

피치 확장 원칙: 지표 정의는 피치 불변, 수치 테이블만 피치별 시트로 추가 (본 문서 = 140µm 시트).

## 1. Common Core 물리 지표

### XDET-EV-101 — DQE @ RQA5 [B]

문헌 베이스라인 (CsI급): peak DQE 0.60~0.64, 주파수별 0.52 (1 lp/mm) / 0.35 (2 lp/mm) / 0.21 (3 lp/mm).

| 항목 | min | typ | max |
|---|---|---|---|
| Peak DQE | 0.55 | 0.60 | 0.65 |
| DQE(1 lp/mm) | 0.45 | 0.52 | 0.58 |

| 알고리즘 적용 후 DQE 변화 | 열화 ≤2% | 열화 0% | (SW 상승 주장은 측정오류 의심 — 별도 검증) |

| 항목 | min | typ | max |

## 6. Caveats

- 본 수치의 베이스라인은 문헌(경쟁사 실측·표준) 기반 — [B] 항목의 계약·대외 사용은 자사 실측 후.
- GOS/타 피치 시트는 차기 패널 확정 시 추가.
- ADR·DL 항목은 잠정치 유지.
