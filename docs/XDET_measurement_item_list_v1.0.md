# XDET 실측 제출 항목 리스트 v1.0 (사람 수행 측정 작업 지시서)

대상: 현 제품 패널 (CsI, 140µm, 3072×3072/3072×2560). 목적: ① 베이스라인 확보(EVAL [B] 치환) ② 골든 데이터셋(GDS) 원료 취득. 모든 데이터는 raw 16-bit, 보정 미적용 원본 + 취득 조건 메타데이터(kV, mA·s, 필터, 선량, SID, 패널 온도, 시각) 필수 기록.

## 운용 규칙

1. 제출 = GDS 레포 업로드 + 메타데이터 시트 작성 (건별). 누락 메타데이터 데이터는 무효 처리.
2. 우선순위: **A → B1~B4 (DQE/MTF 산출 최소 세트) → B5~B11 → C → D**. A+B1~B4 완료 시점에 EVAL [B] 1차 치환 가능.
3. 반복 취득 매수는 최소치 — 이상치 발생 시 추가.
4. 특허 검토는 본 측정과 무관 — 최후순위로 이연 (사용자 지시).
