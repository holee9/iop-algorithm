# XDET 영상 취득 세트 단계별 정리 v1.1 (실측제출항목리스트 v1.0 대체 요약판)

원칙: 데이터는 소비처가 생기는 시점 직전에 취득. 베이스라인 실측은 pass/fail 시험이 아니라 알고리즘 설계 입력.

## 1단계 — 지금 필요 (문서 [B] 치환 + 개발 착수 입력)

| # | 영상 | 조건 | 매수 | 소비처 |
|---|---|---|---|---|
| 1-1 | 다크 | 무조사, 안정화 후 연속 취득 | 10매 | offset 통계, bad pixel, line noise (EV-101/103/105 베이스라인) |
| 1-2 | 브라이트 | 균일 조사, 포화의 40~70% 레벨 | 10매 | gain 맵, NPS, bad pixel (EV-101/103/105) |
| 1-3 | 선량 계단 | 1-2의 1/4, 1/2배 (1배는 1-2 겸용) | 단계당 3매 (+6매) | 선형성, Poisson-Gaussian 노이즈 모델 (FR-C002/C010 파라미터) |
| 1-4 | 엣지 | 납/스틸/구리 후판, 패널 밀착, 거의 수평·수직 + 2~3° 미세 경사 (45° 대각선 금지, 알루미늄 금지) | 방향당 1~2매 (수평+수직) | MTF/SRb 베이스라인 (EV-102) |

합계 약 30매, 반나절 작업. **완료 시 EVAL v1.2 [B] 1차 치환 + SWR 착수 입력 확보.**

## 2단계 — P1 초중반 (해당 WP 착수 전까지)

| # | 영상 | 조건 | 매수 | 소비처 (착수 트리거) |
|---|---|---|---|---|
| 2-1 | Lag 시퀀스 | 강한 노출(정상 2~4배) 직후 연속 다크, 프레임 간격 기록 필수 | 5~10매 × 2~3회 | FR-C005 IRF 파라미터 (WP2 착수 전) |
| 2-2 | Ghost 시퀀스 | 납판 반차폐 강노출 → 균일 브라이트를 직후/1분/5분 시간 계단 | 시점당 1~2매 | FR-C006 설계 (WP2) |
| 2-3 | Grid 장착 세트 | 보유 grid 종류별 — 가능하면 밀도 3부류(36 lines/cm 미만·근접·초과) 포함(aliasing 검증), 정렬/미세 경사 포함 | grid당 5매 | FR-M006/SWR-1006 (WP8), TC-015 |
| 2-4 | 구조물 세트 | 금속 막대/판 포함 촬영 | 배치 3종 × 5매 | FR-C007 오보정 방지 (WP3), TC-007 |
| 2-5 | 포화 시나리오 | 직접선 포함 고선량, 경계 포함 구도 | 3구도 × 5매 | FR-C008 (WP4), TC-008 |
| 2-6 | 기하 격자 팬텀 | 격자 팬텀 정밀 배치 | 3위치 × 3매 | FR-C009 (WP4), TC-009 |
| 2-7 | Scatter 팬텀 | 아크릴/물 두께 계단, grid 유/무 쌍 | 조건당 5매 | FR-M008 커널 설계 (WP9), TC-017 |
| 2-8 | 임상 모사 팬텀 | 보유 팬텀, 표준+저선량 계단 | 팬텀·선량당 5매 | 노이즈 저감 검증 (WP5), 자동윈도우 (WP7), GDS 임상 세트 |
| 2-9 | NDT 시편 + IQI | 용접 시편, duplex wire+단선 IQI, 적산용 연속 프레임 포함 | 조건 3점 × 5매 | WP10, TC-018 |
| 2-10 | Step wedge / 두께 시편 | 대표 재질 | 조건당 5매 | FR-N002 (WP10), TC-019 |

취득 시점 원칙: 각 항목은 해당 WP 착수 직전 — 일괄 취득 불요.

## 3단계 — P1 완료 후 (인허가·공식 성적서)

| # | 항목 | 조건 | 비고 |
|---|---|---|---|
| 3-1 | 공식 선질 셋업 | RQA5 확립: Al 21mm (순도 99.0% type-1100), HVL 실측 kV 확정, SID ≥1.5m, 교정 이온챔버, XN 확정 | 셋업 자체가 산출물 (성적서 전제) |
| 3-2 | 정식 텅스텐 엣지 구매 | W 1~2mm 규격품 | 1단계 대체 엣지의 공식 대체 |
| 3-3 | DQE/MTF/NPS full 세트 | 3선량(XN/2, XN, 2XN) × 균일 16매 + 엣지, 보정 전/후 각각 | IEC 62220-1 성적서급, EVAL v1.x 공식 베이스라인 |
| 3-4 | Bad pixel 공식 매핑 | dark/flat 각 50매 스택 | ASTM E2597 7종 분류 공식판 |
| 3-5 | Lag/Ghost 공식 측정 | ASTM 절차 준용, 반복 통계 | EV-104 공식치 |
| 3-6 | CDRAD/관찰자 연구 세트 | 인허가 프로토콜 확정 후 | VV-010 (유일 인력 시험) |
| 3-7 | NDT 공식 세트 | BAM5급 시편, ASTM/ISO 조건 완비 | 대외 스펙·고객 제출용 |

## 공통 규칙

- 전 영상 raw 16-bit 원본 + 메타데이터(kV, mA·s 또는 선량, 해상도, 다크는 노출 이력, lag는 프레임 간격) 필수.
- 1단계는 정밀 선질 셋업 불요 — 현재 장비 세팅 그대로, 조건 기록만 정확히.
- 패널 형상 변경(신틸레이터 로트, ROIC 설정) 발생 시 해당 단계 재취득.
