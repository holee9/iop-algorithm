# XDET 측정 프로토콜 v1.1 (자동 검증 파이프라인 사양)

목적: EVAL v1.1 지표의 측정 절차를 표준 원문 근거로 규정. 본 문서가 P1 WP11(자동 검증 파이프라인)의 구현 사양이며, VV/TC의 측정 방법 단일 출처이다.

## 개정 이력 (HISTORY)

- v1.1 (issue #34): 빔질을 2계층으로 층화 — **계측 참조 빔질**(RQA5, DQE/MTF/NPS 도메인 독립 비교 기준)과 **응용 빔질**(도메인별). §1 제목을 "의료·계측 참조 빔질"로 명확화, RQA5 등가 대안 필터(`0.5 mm Cu + 2 mm Al ≡ Al 21 mm`, Nowik 2015) 행 추가, **§1b NDT 응용 빔질** 신설, §4 측정 조건 메타에 도메인·빔질 식별자 필수화. 의료 계층(type-1100, Ranger 각주 등) 기존 내용은 전부 보존(추가 편집). 파일명은 유지(문서 맵·RTM 참조 보존).
- v1.0: 최초 발행.

## 1. 의료·계측 참조 빔질 측정 (IEC 62220-1 계열)

> **빔질 2계층 원칙**: 본 §1은 **계측(metrology) 참조 빔질** 계층이다. DQE/MTF/NPS 등 검출기 고유 성능 지표는 고정된 참조 빔질(RQA5)에서 측정하여 **도메인(의료/NDT)에 무관하게 검출기 간 비교 가능성**을 확보한다. 오프셋/게인/에너지 응답 등 **응용(application) 빔질** 계층은 도메인별로 다르며, 의료는 본 §1, NDT는 §1b(ASTM E2597 계열)를 따른다.

### 1.1 공통 셋업

| 항목 | 규정 | 근거 |
|---|---|---|
| 선질 | RQA5: 70~74 kV + 부가필터 Al 21 mm (HVL 기준으로 kV 미세조정) | IEC 62220-1 |
| 선질 대안 | RQA5 등가 대안: **`0.5 mm Cu + 2 mm Al ≡ Al 21 mm`** — 두께·조달 부담↓ 실무 대안 (RQA5 빔질 검증 등가) | Nowik, Physica Medica 2015 |
| Al 필터 순도 | **99.0% (type-1100) 사용** — 초고순도(99.999%)는 저주파 mottle 유발 보고 있음. RQA/HVL 검증은 명목상 99.9% 순 Al(Alloy 1190)을 요구하나 **Alloy 1100은 RQA 빔질의 허용 대안**임 | Ranger et al., Med Phys 2005; IEC 61267 |
| SID | ≥ 1.5 m | IEC 62220-1 (일반촬영) |
| 조사야 | 검출기면 약 16×16 cm, 다단 납 조리개 | IEC 표준 기하 |
| 선량계 | 교정된 이온챔버 (검출기면 air kerma, 후방산란 제외) | IEC 62220-1 |
| 선량 수준 | XN/2, XN, 2XN 3점 (XN: 통상 8.73 µGy 참조, 자사 정격으로 확정) | 문헌 관행 |
| 영상 | raw 16-bit, grid 미장착, offset/gain 보정 상태 정의 명시 (보정 전/후 각각 측정) | IEC + EVAL 요구 |

### 1.2 MTF (edge method)

- Test device: 불투과 edge — W 2 mm 슬랩 (IEC 62220-1 규정형), 검출기면 밀착, 화소 격자 대비 1.5~3° 경사 배치.
- 처리: 자동 edge 각도 추정 → oversampled ESF → LSF(미분+창함수) → FFT → presampled MTF. 스크립트 산출, 인력 판독 없음.
- 판정 연결: EV-102 (MTF@3.57 lp/mm, 알고리즘 전후 유지율).

### 1.3 NPS / NNPS

- 균일 조사 영상 다수 매 확보, 중앙 영역에서 256×256 ROI 다중 추출(반중첩), 트렌드 제거 후 2D FFT 평균 → 1D 축방향 추출(중심축 제외).
- 판정 연결: EV-101 (DQE 산출 입력), 라인노이즈 스펙트럼 성분 정량(EV-105 보조).

### 1.4 DQE

- DQE(f) = MTF²(f) · q · Ka / NPS(f). q(RQA5 광자 fluence 계수)는 IEC 표 값 사용.
- 3개 선량 수준 × 알고리즘 전/후 → EV-101 판정 자동화.

### 1.5 Lag / Ghost

- ASTM E2597 lag 절차 준용(의료·NDT 공용): 포화 근접 노출 후 프레임 시퀀스 취득, first-frame lag % 자동 산출. Ghost는 고대비 패턴 노출 후 균일 조사에서 잔상 CNR 자동 검출.
- 판정 연결: EV-104.

### 1.6 관찰자 대체 자동 IQA (개발 루프)

- CDRAD 팬텀 자동 스코어링(IQFinv), 골든 데이터셋 임상 모사 영상의 무참조 IQA 지표 병용.
- 인허가용 관찰자 연구(VV-010)는 별도 프로토콜(인원·판독 조건·통계) — 인허가 단계에서 상세화.

## 1b. NDT 응용 빔질 (ASTM E2597/E2698, ISO 17636-2)

> **NDT는 RQA를 사용하지 않으며, ASTM E2597이 정의하는 자체 표준 빔 스펙트럼과 팬텀(step wedge)을 사용한다.** RQA5(§1)는 검출기 고유 성능의 **계측 참조 빔질**로만 유지되고, NDT의 offset/gain·에너지 응답·효율 특성은 아래 **응용 빔질**에서 측정한다. (본 §1b는 빔질·스펙트럼·여과 셋업을 규정하고, 산출 지표의 계산 절차는 §2와 공용이다.)

### 1b.1 응용 빔질 셋업

| 항목 | 규정 | 근거 |
|---|---|---|
| 표준 빔질 | ASTM E2597 표준 방사 스펙트럼 — **다중 에너지·다중 빔질**에서 특성화 (RQA 아님) | ASTM E2597/E2597M-22 |
| 관전압/선원 | 고에너지 X선 최대 ~450~500 kVp, 또는 감마선(Ir-192 / Co-60) | ASTM E2597, ISO 17636-2 |
| 경화(여과) 필터 | **Cu 빔 경화 필터(수십 µm ~ 수 mm, kV·강재 두께에 비례 증가)**, 필요시 Sn/Pb 추가 | ASTM E2597 / ISO 17636-2 |
| Pb 스크린 | 0.02~0.2 mm (최대 500 kV) | ISO 17636-2 |
| 재질 교정 | **강/알루미늄 step wedge** (Al 팬텀 아님) — SMTR·CSa 산출 기준 | ASTM E2597 |
| DDA 시험 규격 | ASTM E2698(DDA 방사선 검사)과 짝을 이룸 | ASTM E2698 |

### 1b.2 응용 빔질 특성 지표 (ASTM E2597)

빔질별로 아래 항목을 산출한다 (계산 절차는 §2와 공용, 판정 기준은 EVAL EV-301/303).

| 지표 | 정의 | 비고 |
|---|---|---|
| iSRb | 보간 기본 공간분해능(interpolated basic spatial resolution) | duplex wire 기반 |
| **dSNRn @ 1 mGy** | 검출기 SNR 정규화값(1 mGy 기준) — NDT 응용 계층 효율 지표 | EV-301 확장 판정 |
| CSa | 달성 대비 감도(achievable contrast sensitivity) | step wedge 기반 |
| SMTR | 특정 재질 두께 범위(specific material thickness range) | 대표 재질별 (EV-303) |
| image lag | 프레임 잔상 | §1.5 lag 절차 공용 |
| burn-in | 잔류 번인 | 장기 노출 |
| bad pixel | 불량 화소 분류 | E2597 분류 (§2 공용) |
| ISR | 내부 산란 방사(internal scatter radiation) | 검출기 내부 산란 |

### 1b.3 [QUARANTINE] 실측·피팅 경계

> **본 §1b의 NDT 응용 빔질 및 dSNRn@1mGy 요구는 규범(normative) 사양일 뿐이며, 샘플 세트(issue #29)로부터 수치를 역산·피팅하지 않는다. NDT 빔질 실측은 유도 취득 세트(issue #33) 도착 후 수행한다.**

## 2. NDT 지표 측정 (ISO 17636-2 / ASTM E2597)

| 지표 | 절차 | 자동화 |
|---|---|---|
| SNRn | 균일 영역 SNR 측정 → SRb로 정규화 (SNRn = SNR × 88.6/SRb[µm]) | 스크립트 |
| SRb | Duplex wire IQI 영상 → 20% dip 기준 자동 판독 | 자동 판독 SW |
| 단선 IQI | Wire IQI 자동 검출·판독 | 자동 |
| CSa/SMTR | ASTM E2597 step-wedge 절차 | 스크립트 |
| Bad pixel | ASTM E2597 7종 분류 매핑 (dark/flat 스택 통계) | 스크립트 |

## 3. 성능 측정 (EV-401/402)

- 처리시간: 3072×3072 16-bit 프레임, 파이프라인 단계별 타이머, cold/warm 분리, 100회 중앙값.
- 티어 동일성: 동일 입력 → CPU/GPU 출력 diff — 결정론 기능은 bit 비교, 부동소수점 기능은 정의된 허용오차(사양서에서 확정) 비교.

## 4. 데이터·형상 규칙

- 모든 측정 raw + 산출물은 골든 데이터셋 레포에 버전 고정. 측정 조건 메타데이터(kV, 필터, 선량, 온도)에 더해 **도메인(의료/NDT)·빔질(beam quality) 식별자**를 필수 메타로 기록 — 계측 참조 빔질(RQA5, §1)과 응용 빔질(§1b)을 명확히 구분하기 위함.
- 프로토콜 개정은 EVAL과 동기 개정 (측정법 변경 = 지표 재베이스라인).

## 5. Caveats

- RQA5의 kV는 장비별 HVL 실측으로 70~74 kV 범위에서 확정 — 명목 kV 고정 금지.
- XN 값은 자사 임상 정격 확정 후 프로토콜 v1.1에 명기.
- IEC 62220-1 최신판(개정 여부) 원문 구매·대조 후 세부 수치(조리개 위치 등) 최종 확정 필요.
